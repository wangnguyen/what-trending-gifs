># Simple app to get Trending Gifs

## For dev:
* Install packages: `npm install`
* Run cmd: `npm run dev`

## Build production: 
* run cmd: `npm run build`
* **Node:** all files will be compile in `./dist/` direactory

## Libraries
* `React`
* `Webpack`
* `Redux`
* `Redux Saga`
* `Material UI`
* `JSS`
